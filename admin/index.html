Analytics & Marketing Pixels

Connect Meta Pixel, Google Analytics, and Meta Conversion API — no code needed

Meta Pixel (Facebook Ads)

Tracks visitors and conversions from your Facebook & Instagram ad campaigns. Required for retargeting audiences.

Meta Pixel
Not connected
Paste your Pixel ID above to see the generated tag
  • Go to Meta Business Manager → Events Manager
  • Click Add Data Source → Web → Meta Pixel
  • Copy the Pixel ID (16-digit number) and paste above
  • Click Save Pixel Settings below

Google Analytics 4

Track website traffic, user behavior, and conversion goals from your Google Analytics account.

Google Analytics 4
Not connected
Paste your Measurement ID above to see the generated tag
  • Go to analytics.google.com → Admin
  • Click Create Property → choose Web
  • Copy the Measurement ID (starts with G-) and paste above
  • Click Save GA Settings below

Meta Conversions API (CAPI) — Server Side

Send conversion events directly from your server to Meta — bypasses ad blockers and iOS restrictions. More accurate than Pixel alone. Works alongside your Pixel for maximum data quality.

Conversions API
Not connected
  • In Meta Events Manager → Data Sources → your Pixel
  • Go to Settings → Conversions API
  • Click "Generate access token" and copy it
  • Paste the token above and click Save — lead events will be sent server-side automatically
